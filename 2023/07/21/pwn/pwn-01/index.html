其中

  • Stack 栈区

  • Data 为数据段，存放由程序员初始化的全局/静态变量

2.栈区

每个栈内按照：

  • 参数（倒序）

  • 返回地址

  • 前帧指针

  • 局部变量

的顺序由高地址到低地址排列

Return Address 返回地址 里面存放了一个地址，在该区域弹出栈时，被eip寄存器接收，使该地址成为程序下一步执行的位置。他决定了函数返回时程序将跳转至何处执行。通过修改返回地址的数值，将错误的值输入eip寄存器便是基本的攻击原理。

前帧指针 个人理解用于指向父函数的前帧指针，并且在运行至该函数时由ebp指向他用于在改栈内定位，通过偏移量获取目标内容
